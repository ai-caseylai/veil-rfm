# 《老教授與菜鳥的十四堂 RFM 課》

> 漫畫劇本 · 改編自 Veil RFM 綜合問答手冊
> 風格致敬《最後 14 堂星期二的課》(Tuesdays with Morrie)
> 角色：莫教授（老教授，68 歲，RFM 分析領域泰斗）/ 小林（菜鳥資料分析師，26 歲）

---

## 角色設定

| | 莫教授 | 小林 |
|---|---|---|
| 年齡 | 68 | 26 |
| 身份 | 零售分析學退休教授 | 剛入職的資料分析師 |
| 個性 | 睿智、溫暖、愛用比喻 | 好學、急躁、容易焦慮 |
| 口頭禪 | 「資料是活的，你要聽它說話。」 | 「教授，我不懂⋯⋯」 |
| 服裝 | 老花眼鏡、素色襯衫、總是帶著一杯茶 | 格子襯衫、牛仔褲、MacBook 不離手 |
| 場景 | 莫教授的書房，滿牆的書和資料圖表 | 同一書房，坐在沙發上做筆記 |

## 第一課：初見 — 什麼是 RFM？

```
[畫面：一間充滿書香的書房。夕陽透過窗簾灑進來。
莫教授坐在藤椅上，手裡捧著一杯熱茶。小林緊張地坐在對面沙發上。]

小林：（緊張地搓手）
「莫教授，謝謝您願意見我。我剛入職一週，
老闆叫我做 RFM 分析，我完全不知道那是什麼⋯⋯」

莫教授：（微微一笑，放下茶杯）
「小林啊，你吃過樓下的牛肉麵嗎？」

小林：（愣住）
「吃⋯⋯吃過啊。上週吃了三次。」

莫教授：
「那老闆認得你了嗎？」

小林：
「認得了！現在遠遠看到我就喊『小帥哥，今天要大碗的？』」

莫教授：（眼睛發亮）
「這就是了。你上週才來過（Recency 高），
一週來三次（Frequency 高），
每次都點大碗加滷蛋（Monetary 高）。
在牛肉麵老闆眼中，你就是他的 Best Customer。」

小林：
「所以 RFM 是⋯⋯」

莫教授：
「R，Recency，你上次來的時間——越近越好。
F，Frequency，你來的次數——越多越好。
M，Monetary，你花的錢——老闆當然希望你點大碗的。」
「把這三個維度各打 1 到 5 分，
三項都滿分的就是你——Best Customer 555。」

小林：（恍然大悟）
「原來這麼簡單！」

莫教授：（搖頭）
「簡單？下一課我們再聊。現在，先喝茶。」
```

## 第二課：十一張臉 — 認識你的客戶

```
[畫面：莫教授從書架上拿出一疊手繪的人物卡片，共 11 張。]

莫教授：
「昨天我睡不著，畫了這個。」

小林：（拿起第一張，上面畫著一個穿西裝、戴皇冠的人）
「Best Customers⋯⋯國王？」

莫教授：
「VIP。每次來都坐最好的位子、點最貴的菜。
你覺得該怎麼對他？」

小林：
「當然是好好伺候啊！不能讓他跑掉！」

莫教授：（點頭）
「對。那這個呢？」

[小林拿起第二張：畫著一個穿圍裙的人，手裡提著菜籃]

小林：
「Loyal Customers⋯⋯家庭主婦？」

莫教授：
「忠誠、穩定、每週固定來兩次。
不會跑，但你要給她一個跑的理由——『累積點數換免費菜』。
別讓她覺得理所當然。」

[小林一一翻閱剩下九張卡片]

莫教授：
「第三張，Potential Loyalist——那個剛搬家到你店附近的新鄰居，
還在試探期，多對他笑一笑。」
「第四張，Low-spending Active Loyal——每天都來，
只買一顆茶葉蛋。你可以試試問他『今天要不要順便帶杯豆漿？』」
「第五張，High-spending New——第一次來就點滿漢全席的怪客。
他可能只是路過。不跟進，下週就忘了你。」
「第六張，Almost Lost——以前常來的老客人，
最近不來了。你不打電話問，他就真的走了。」
「第七張，Churned Best——你以前最愛的老客人，已經三個月沒見。
他可能去了對面新開的店。你要用最大的誠意去挽回。」
「第八張到第十一張⋯⋯」（嘆氣）
「是那些你已經快忘記，或真的忘了的客人。」

小林：（沉默了一會）
「教授，您畫這些的時候⋯⋯在想什麼？」

莫教授：
「我在想，每一個數字背後，都是一個人。」
「資料分析不是算數字，是聽故事。」
```

## 第三課：數學的溫柔 — 分數怎麼來？

```
[畫面：白板上畫滿了百分位數的公式。小林皺著眉頭。]

小林：
「教授，percentile rank 到底是什麼⋯⋯」

莫教授：
「你小時候考試，全班 40 人，你考第 3 名，
你的百分位數是多少？」

小林：
「呃⋯⋯(40-3)/40 = 92.5%？」

莫教授：
「對。RFM 也是一樣。你不是跟滿分比，
你是跟其他客戶比。」
「你買了 10 次，覺得很多？
但如果其他人平均買了 20 次，你就只有 2 分。」
「你花了 $100？如果其他人平均花 $500，你也只有 1 分。」

小林：
「所以 RFM 是一個⋯⋯相對的概念？」

莫教授：（滿意地笑）
「你終於懂了。資料是活的。
它不是絕對值，是你的客戶們一起投票決定的。」
```

## 第四課：排行榜的傲慢

```
[畫面：小林看著螢幕上的排行榜，眼睛發光。]

小林：
「教授！C004 好厲害！$7,130，16 筆訂單，RFM 555！
她一個人貢獻了 60% 的營收！」

莫教授：（放下茶杯，表情嚴肅）
「你知道這代表什麼嗎？」

小林：
「代表她是最重要的客戶？」

莫教授：
「代表你們公司岌岌可危。」
「如果她明天不買了呢？後天呢？」
「你 60% 的營收押在一個人身上。
這不是成功，這是風險。」

小林：（愣住）
「那⋯⋯怎麼辦？」

莫教授：
「你看第二名的 C002——$2,005，10 筆訂單。
第三名的 C001——$1,456，12 筆。
還有三個 Potential Loyalist。」
「你要做的不是崇拜第一名，
而是讓第二三四五六名都變強。」

小林：
「就像⋯⋯不能只養一頭金牛？」

莫教授：
「對。牧場裡要多養幾頭。」
```

## 第五課：如果 — 情境模擬的魔法

```
[畫面：小林興奮地操作著 What-If 面板。]

小林：
「教授教授！如果 C008——那個 Lost Cheap——
買了 5 次、花了 $200 每次⋯⋯」

莫教授：
「她會從 111 變成什麼？」

小林：（按下模擬鍵）
「⋯⋯Potential Loyalist！」

莫教授：
「你看，這就是資料的魔法。」
「它告訴你：沒有人是不可救藥的。」

小林：
「但 C008 已經 84 天沒買了⋯⋯」

莫教授：
「你有沒有想過，為什麼是 84 天？」

小林：
「呃⋯⋯因為她不想買？」

莫教授：
「也可能是她搬家了、生病了、
或者根本不知道你們開了線上商店。」
「資料告訴你『什麼』，但不會告訴你『為什麼』。」
「『為什麼』需要你去問。」

小林：（若有所思）
「所以 What-If 不只是模擬數字改變⋯⋯」

莫教授：
「它是模擬『如果你真的去關心一個客戶，會發生什麼』。」
```

## 第六課：告別 — 流失的真相

```
[畫面：窗外下著雨。茶涼了。]

小林：
「教授，C002 已經 22 天沒買了。
她以前花那麼多錢⋯⋯為什麼不來了？」

莫教授：（望著窗外）
「你談過戀愛嗎？」

小林：（臉紅）
「呃⋯⋯有。」

莫教授：
「分手的時候，你不會在前一天晚上通知對方吧。」

小林：
「⋯⋯不會。」

莫教授：
「客戶也一樣。她不會告訴你『我要離開了』。」
「她只是⋯⋯慢慢地不來了。」
「第 1 天沒來，你沒發現。第 7 天沒來，你覺得還好。
第 14 天沒來，你開始緊張。第 22 天⋯⋯」
「她已經在對面那家店辦會員卡了。」

小林：（急切地）
「所以我們要在第幾天的時候行動？」

莫教授：
「不是第幾天。是她還在的時候。」
「Recency = 4 的時候你不打電話，
等到 Recency = 1，她已經忘記你店名了。」

小林：（低下頭）
「那我現在該對 C002 說什麼？」

莫教授：
「不是『我們最近有優惠喔』。
是『好久不見，我們很想你。你最近還好嗎？』」
```

## 第七課：錢的重量 — 營收的真相

```
[畫面：莫教授在茶几上排硬幣。一個大金幣，幾個銀幣，幾個銅幣。]

莫教授：
「這是你們公司的營收地圖。」

小林：（指著最大的金幣）
「C004⋯⋯」

莫教授：（用放大鏡照著金幣）
「你看到什麼？」

小林：
「一個很大的客戶。」

莫教授：（把放大鏡遞給小林）
「再仔細看。」

小林：（湊近看）
「⋯⋯上面有裂痕？」

莫教授：
「集中風險。一個客戶拿走 60% 營收。」
「你們不是在經營一個客戶群，
是在供奉一尊神。」
「神走了，廟就空了。」

小林：
「那要怎麼分散⋯⋯」

莫教授：（把幾個銀幣推到中間）
「你的三個 Potential Loyalist——C001、C002、C005。」
「如果每個人的消費提升 30%⋯⋯」
「你就從一神教，變成多神教。」

小林：（笑出來）
「教授，您的比喻⋯⋯」

莫教授：
「資料不神聖，客戶才神聖。」
「把雞蛋放在不同籃子裡——這是祖母的智慧，不是大數據。」

## 第八課：對比 — 學會欣賞差異

```
[畫面：莫教授拿出兩片葉子。]

莫教授：
「哪片比較好？」

小林：
「左邊這片比較大、比較綠⋯⋯」

莫教授：
「C004 和 C001？」

小林：（看資料）
「呃，C004 明顯比較好。分數、消費、頻率都贏⋯⋯」

莫教授：
「但 C001 在成長。她從 6 筆變成 12 筆。
你看到了嗎？不是現在的數字，是趨勢。」
「C004 是已經登頂的山。
C001 是還在爬的山。」
「你不能只欣賞山頂的風景。」

小林：
「所以客戶對比不是比高低⋯⋯」

莫教授：
「是比故事。每個客戶有不同的故事線。
你的工作是讀懂每個故事，而不是給他們排名。」
```

## 第九課：尋找 — 篩選的藝術

```
[畫面：小林在螢幕前輸入各種篩選條件。]

小林：
「訂單 >10、消費 >$1,000、recency <30⋯⋯
只有兩個人符合！」
「教授，篩選條件是不是太嚴格了？」

莫教授：
「你為什麼要篩選？」

小林：
「因為⋯⋯老闆叫我找出最好的客戶？」

莫教授：（搖頭）
「你是在找『符合你條件』的客戶，
而不是『真正需要你』的客戶。」
「試試這些條件：訂單 = 1、消費 >$100。」

小林：（輸入）
「⋯⋯C006？只有一筆訂單，花了 $200。」

莫教授：
「她為什麼只買一次？
是產品不好？是價格太貴？是忘記你們了？」
「她需要你，比 C004 更需要你。」
「C004 會自己來。C006 不會。」

小林：
「所以我篩選時應該⋯⋯」

莫教授：
「不要找『你最愛的客戶』。
找『最需要你的客戶』。」
```

## 第十課：新與舊 — 時間的朋友

```
[畫面：莫教授翻著一本舊相簿。]

莫教授：
「你覺得，老朋友和新朋友，哪個重要？」

小林：
「都重要？」

莫教授：
「你只有 $100 預算。」
「老朋友（回購客戶）平均消費 $1,935。
新朋友（新客戶）平均消費 $125。」
「你要把錢花在哪？」

小林：
「⋯⋯老朋友？」

莫教授：
「但不是不交新朋友。」
「你看 C006——新客戶，花了 $200。
她只是還沒回來。」
「新客戶是種子，回購客戶是果實。」
「你要澆水給種子，但靠果實吃飯。」

小林：
「所以⋯⋯75% 精力給回購客戶，25% 給新客戶？」

莫教授：
「比例你自己抓。但記住：
一個老朋友的回頭，勝過十個新朋友的試探。」
```

莫教授：

「這座城市，就是你客戶的未來地圖。」

## 第十二課：遷移 — 客戶的旅行

```
[畫面：莫教授在茶几上鋪開一張手繪地圖。
上面有 11 個區域，用箭頭連線。]

莫教授：
「這是我花了四十年的心血——
客戶遷移地圖。」

小林：（仔細看著箭頭）
「所以客戶不會停在一個地方⋯⋯」

莫教授：
「對。今天是 Best Customers，
下個月可能變成 Almost Lost。」
「這條紅色的箭頭，從 Best 指向 Churned——」
「就是你們公司的 C004，如果有一天不來了。」

小林：（緊張）
「那⋯⋯綠色箭頭呢？」

莫教授：
「Hibernating → Potential Loyalist。
這是最美的旅程。」
「一個已經快睡著的客戶，因為你的一通電話、
一張優惠券、一句問候——」
「他醒了，開始買，變成忠誠客戶。」
「這條路很少人走，但走通了的，都是傳奇。」

小林：
「教授，哪條路最難走？」

莫教授：（手指移到地圖的右下角）
「Lost Cheap → 任何地方。」
「一旦掉到這裡，就像流沙。
不是不能救，是救了之後的代價，
可能比開發新客戶還高。」

小林：（靜靜地看著地圖）
「所以我們要在他們掉進去之前⋯⋯」

莫教授：
「對。在 Almost Lost 的時候。
那是黃金救援時間。」
```

## 第十四課：最後一堂 — 資料的靈魂

```
[畫面：夕陽西下。書房裡的光線變得柔和。
茶壺已經空了。小林的眼眶有點紅。]

莫教授：
「小林，你覺得 RFM 三個字母，哪個最重要？」

小林：
「Recency？越近越好⋯⋯」

莫教授：（搖頭）
「不。」

小林：
「那⋯⋯Frequency？越多越好？」

莫教授：（再搖頭）
「也不對。」

小林：
「Monetary？⋯⋯」

莫教授：
「是 Human。」
「不是 R、不是 F、不是 M。是 H。」

小林：（困惑）
「可是⋯⋯RFM 沒有 H 啊？」

莫教授：
「我教書教了四十年。
到最後我才發現，
Best Customers 不一定是花最多錢的客戶。」
「他們是那些會在 Facebook 上幫你推薦的人。
是那些寫 email 說『你們的產品改變了我的生活』的人。
是那些你生病時會寄慰問卡的人——即使他們三個月沒買。」

小林：（沉默了很久）
「⋯⋯教授，我明白了。」

莫教授：（站起身，拍拍小林的肩膀）
「不，你才剛開始。」
「這十四堂課，只是入門。」
「真正的大數據，在你的心裡。」
「機器可以告訴你誰買了什麼。
但只有你，可以問『為什麼』。
只有你，可以打那通電話。
只有你，可以在對方說『我搬家了』的時候，
說『沒關係，我們可以幫你轉到附近的店』。」

[畫面：小林走出書房。回頭看了一眼。]

小林：
「教授⋯⋯我下週還可以來嗎？」

莫教授：（站在門口，夕陽在他身後）
「隨時。這是我家的地址——」
「不是 email，不是 LINE。是地址。」
「因為有些東西，要靠人傳，不能靠資料傳。」

[最後一格：莫教授回到書房，拿起桌上的老花眼鏡，
看著小林留下的筆記本。筆記本第一頁寫著：
「H = Human。資料的靈魂。」]

莫教授：（對著空房間自言自語）
「⋯⋯這個學生，會比我厲害。」
